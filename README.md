# 🔐 Quantum Vault: Interactive Quantum Search Portal

This project is a web application developed by a 12th-grade student that performs real-time quantum searching on **IBM Torino (Italy)** quantum hardware.

## 🚀 About the Project
Quantum Vault brings the complex world of quantum computing to a user-friendly interface. Users select a number (0-7), and the system constructs a dynamic quantum circuit using **Grover's Algorithm** to locate that target in quantum space.

## 🛠️ Tech Stack
* **Quantum Software:** Qiskit, Qiskit Runtime Service
* **Quantum Hardware:** IBM Torino (Italy)
* **Web Interface:** Streamlit
* **Localization:** Turkish & English (with Preference Memory)

## 🏗️ How It Works
1. The user defines a target number in a 3-qubit computational space.
2. The app generates an **Oracle** to mark the target and a **Diffuser** to amplify the probability amplitude.
3. The circuit is transpiled and executed on the IBM hardware in Italy (`ibm_torino`).
4. Results are returned to the user as a probability-based histogram.

# 🔐 Quantum Vault: Etkileşimli Kuantum Arama Portalı

Bu proje, bir 12. sınıf öğrencisi tarafından geliştirilmiş, **IBM Torino (İtalya)** kuantum donanımı üzerinde gerçek zamanlı arama yapan bir web uygulamasıdır.

## 🚀 Proje Hakkında
Quantum Vault, kuantum bilgisayarların karmaşık dünyasını kullanıcı dostu bir arayüze taşır. Kullanıcılar bir sayı seçer (0-7) ve sistem bu sayıyı bulmak için **Grover Algoritması** kullanarak dinamik bir kuantum devresi inşa eder.

## 🛠️ Kullanılan Teknolojiler
* **Kuantum Yazılım:** Qiskit, Qiskit Runtime Service
* **Kuantum Donanım:** IBM Torino (İtalya)
* **Web Arayüzü:** Streamlit
* **Dil Desteği:** Türkçe ve İngilizce (Otomatik Hatırlama Özellikli)

## 🏗️ Nasıl Çalışır?
1. Kullanıcı 3-qubitlik bir uzayda hedef sayısını belirler.
2. Uygulama, hedef sayıyı işaretleyen bir **Oracle** ve genliği artıran bir **Diffuser** katmanı oluşturur.
3. Devre, IBM'in İtalya'daki donanımına (`ibm_torino`) gönderilir.
4. Sonuçlar, olasılık tabanlı bir grafik (histogram) olarak kullanıcıya geri döner.
